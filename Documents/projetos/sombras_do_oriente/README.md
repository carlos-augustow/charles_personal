Uma maison de perfumaria de nicho que oferece experiências olfativas personalizadas. Transforme suas memórias em fragrâncias exclusivas.

## ✨ Características

- **Design Elegante**: Layout moderno e responsivo focado na experiência do usuário
- **Perfumaria de Luxo**: Identidade visual sofisticada com paleta de cores exclusiva
- **Experiências Personalizadas**: Páginas dedicadas para experiências femininas e masculinas
- **Interatividade**: Animações suaves com Framer Motion
- **Responsivo**: Funciona perfeitamente em todos os dispositivos
- **Performance**: Otimizado com Next.js 14 e TypeScript

## 🛠️ Como Executar

### Pré-requisitos
- Node.js 18+ 
- npm ou yarn

### Instalação
```bash
# Clone o repositório
git clone [url-do-repositorio]

# Entre no diretório
cd sombras-do-oriente

# Instale as dependências
npm install

# Execute em modo desenvolvimento
npm run dev
```

O projeto estará disponível em `http://localhost:3000`

### Comandos Disponíveis

```bash
# Desenvolvimento
npm run dev

# Build para produção
npm run build

# Executar produção
npm run start

# Lint
npm run lint
```
